# Avocados Store

Built using NextJs framework

NextJs Pros:

- Routing
- Code Splitting
- Assets hashing
- Pre-rendering - SSR
- Prefetching

## Usage

- Clone the repository

  ```
    git clone https://github.com/dmiranda2791/avo-store.git
  ```

- Install dependencies
  ```
    yarn
  ```
- Run in development mode
  ```
    yarn dev
  ```
- Build production bundle
  ```
    yarn build
  ```
- Deploy
